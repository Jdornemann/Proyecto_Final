# Proyecto_Final
Proyecto Final - FlatKeeper

FlatKeeper es un proyecto para administrar propiedades cubriendo todas las aristas involucradas en el negocio, partiendo por requisitos de limpieza, 
disponibilidad, llamado a fontanero, electricistas, compras de insumos. Creado con la intención de lograr la coordinación entre un area y otra como dentro de un hotel.
